
INSTALLATION
------------

L'installation du gestionnaire de protocole "designer:..." est pour
le moment un processus manuel.

Voici comment procéder :

- Ouvrir le dossier C:\Program Files\Epsitec\Crésus Designer (ou
  C:\Program Files (x86)\Epsitec\Crésus Designer sur sys. x64) et y
  copier les fichiers *.exe et *.dll Protocol.exe et App.Designer.-
  Protocol.Contract.dll.

- Exécuter "designer.reg" pour enregistrer le protocole dans le
  registre Windows (ou "designer-x86.reg" sur un système x64).

- Relancer Visual Studio, si celui-ci était ouvert au moment de
  l'étape ci-dessus.

UTILISATION
-----------

- Exécuter Designer.

- Depuis Visual Studio, utiliser CTRL+clic sur les liens débutant
  par "designer:" dans les commentaires, par exemple dans les divers
  fichiers Res.cs.

  Cela bascule dans Designer, sur la ressource correspondante.
