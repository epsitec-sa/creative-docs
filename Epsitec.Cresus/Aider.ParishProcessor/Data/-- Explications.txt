Fichiers présents dans ce dossier, utiles pour le projet :

- 2023-02-xx Clean Parish Info.txt

  Fichier de référence contenant les NPA/rues/numéros associés aux paroisses. Ce
  fichier est consommé par l'utilitaire de conversion. Ce fichier est extrait par
  copier/coller des colonnes A-F du fichier EXCEL correspondant (en sautant la
  1ère ligne).


- 2023-02-xx parish names.txt

  Fichier de référence avec les noms de paroisses, par régions.



- 2023-02-xx parishes.txt

  Fichier produit par l'utilitaire de conversion: mapping NPA/rue/numéro -> paroisse
  Utilisé pour assigner les adresses aux paroisses.
  Utilisé pour initialiser les groupes "paroisses".
