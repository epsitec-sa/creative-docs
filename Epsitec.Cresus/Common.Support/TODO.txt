RESSOURCES DANS LA BASE DE DONNEES
-----------------------------------------------------------------

La base de données des ressources d'une application se nomme :

	application.resdb

(avec éventuellement l'extension ajoutée par DbInfrastructure et
les autres couches).

En plus des tables décrivant les méta-données, il y a la table
suivante :

-	DbTable.Name = "Data";

La table "Data" contient toutes les données, par défaut (non localisées)
et aussi toutes les autres (localisées ou personnalisées).

Les colonnes dans "Data" sont les suivantes :

-	Clef et statut (CR_ID, CR_REV, CR_STAT).
-	Name  (DbColumn.Name = "Name"; ), type DbTypeString, max=100
-	Level (DbColumn.Name = "Level";), type DbTypeString, max=4
-	Data  (DbColumn.Name = "Data"; ), type DbType(DbSimpleType.ByteArray)
	contenant des BLOBs binaires.

La recherche se fait au moyen d'un SELECT tel que :

-	Name   = id de la ressource
-	Level  = niveau (Default="00", Localised="fr", Customised="Xfr")
-	CR_REV = 0
